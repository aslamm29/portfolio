Fundraiser App

View Frontend Code View Backend Code

This application is a website dedicated to raising awareness about new non-profits and has the ability to donate to causes! It is for fundraisers and donators. Features: 1. Login/sign up flow with e-mail address 2. Create a cause and upload photos (multiple, displayed in a carousel) 3. Browse page with filters for: cause, fundraise deadline 4. Direct messaging between users 5. Donating

Tech Stack Used Frontend: HTML, CSS SCSS, JavaScript (ES6+), React Hooks & Backend: Python (Flask) & Database: PostgreSQL
